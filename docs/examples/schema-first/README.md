# Schema-First Approach Example

This example demonstrates the schema-first approach to GraphQL.NET.

## Running the Example
```bash
cd docs/examples/schema-first
dotnet run
```

## What it Shows
- Defining types using `ObjectGraphType`
- Adding fields to types
- Creating a query with arguments
- Using sample data classes

## Key Concepts
1. **Schema-First**: Define your GraphQL schema first, then implement resolvers
2. **Type Safety**: Strongly typed fields ensure compile-time validation
3. **Query Arguments**: Define input parameters for queries
