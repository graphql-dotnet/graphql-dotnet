# Samples

The GraphQL.NET repository ships with several sample projects. Each one lives under `src/`
in the repository root and targets .NET 8.

## Schema-First Demo

**Project:** [`src/GraphQL.SchemaFirstDemo`](https://github.com/graphql-dotnet/graphql-dotnet/tree/master/src/GraphQL.SchemaFirstDemo)

Demonstrates the **Schema-First** (SDL-First) approach. You write the full schema in
[GraphQL SDL](https://graphql.org/learn/schema/) and then wire .NET resolver classes onto it.
